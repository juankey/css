selector de tipo o selector de etiqueta

Selecciona todos los elementos que coinciden con el nombre de nodo,de manera simple, son las mismas etiquetas html

sintaxis: nav, article, main, body...

ejemplo: 


                    nav{
                        color:red;
                    }
                    main{
                        background:green;
                    }

selector de clase

selectores que pueden agrupar a varios elementos seleccionadolos mediante el atributo class(.)

.classname, .principal

ejemplo: 


                        .classname{
                            color:red;
                        }
                        .principal{
                            background:green;
                        }

selector de id

Selecciona un elemento basado en su atributo "id", es un atributo unico. no pudiendo haber otro de su mismo nombre

sintaxis: #class, #betroon

ejemplo: 


                        #class{
                            color:red;
                        }
                        #betroon
                            background:green;
                        }

selector universal

Selecciona todos los elementos, siendo utilizado para resetear los estilos del documento html.

sintaxis: *

ejemplo: 


                        *, *::before, *::after{
                            padding: 0;
                        }

selector de atributo

Selecciona un atributo del documento html

sintaxis: lang, type

ejemplo: 


                            [attr~=value]{
                            padding: 0;
                            }
